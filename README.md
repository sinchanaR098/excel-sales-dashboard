#  Interactive Sales Dashboard (Excel)

##  Project Overview
This project demonstrates an interactive Sales Dashboard created using Microsoft Excel. The dashboard provides insights into sales performance using Pivot Tables, Pivot Charts, and Slicers.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

##  Features
- Sales by Region
- Sales by Product
- Monthly Sales Trend
- Interactive Dashboard
- Dynamic Filtering using Slicers
